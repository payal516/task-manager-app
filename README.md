# Task Manager App

A React app to manage your daily tasks efficiently.

## Features
- Add and delete tasks
- Mark tasks as complete
- Filter by All / Pending / Completed
- Data saved automatically in localStorage

## Tech Used
- React.js
- Tailwind CSS
- Vite
